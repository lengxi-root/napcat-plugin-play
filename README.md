# napcat-plugin-play

NapCat 娱乐插件 - 表情包 / 点歌 / AI绘画

## 指令列表

### 📸 表情包功能

| 指令 | 说明 |
|------|------|
| `meme列表` | 查看表情列表 |
| `表情名@人` | 制作表情 |
| `表情名+详情` | 查看表情用法 |
| `meme搜索+词` | 搜索表情 |
| `随机meme` | 随机生成表情 |
| `meme更新` | 更新表情数据 |

### 🎵 点歌功能

| 指令 | 说明 |
|------|------|
| `点歌+歌名` | 搜索歌曲 |
| `听+序号` | 播放搜索到的歌曲 |

### 🎨 AI绘画功能

| 指令 | 说明 |
|------|------|
| `画+描述` | 文字生成图片 |
| `画+@某人+描述` | 用头像生成图片 |
| `引用图片+画+描述` | 修改图片 |

### ⚙️ 管理功能

| 指令 | 说明 |
|------|------|
| `娱乐菜单` | 显示功能菜单 |
| `设置主人+QQ` | 添加主人 |
| `删除主人+QQ` | 移除主人 |
| `主人列表` | 查看主人列表 |
